Overview

Saralkotha is a web-based room renting system developed as a 7th-semester project for Tribhuvan University (Patan Multiple Campus).
The system aims to make it easier for people to search, rent, and list rooms, flats, or houses online in cities like Kathmandu, Lalitpur, and Bhaktapur.
